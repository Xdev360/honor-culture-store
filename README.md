# HONOR Culture

A premium black & white fitness apparel storefront with a React + Vite frontend and a Node/Express JSON API.

## Tech Stack
- Frontend: React + Vite
- Backend: Node.js + Express
- Styling: Custom CSS, Bodoni Moda typography

## Run Locally

### 1) Backend API
```bash
cd "/Users/xbigbrainnx/Desktop/HONOR CULTURE/backend"
npm install
npm start
```

### 2) Frontend
```bash
cd "/Users/xbigbrainnx/Desktop/HONOR CULTURE/frontend"
npm install
npm run dev
```

Open the frontend URL printed by Vite (usually `http://localhost:5173`).
